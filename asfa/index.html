ASFA - Access System From Anywhere

Introducing ASFA, the ultimate tool developed by Logic Realm that can convert your computer to a cloud computer. You can access your computer world wide without having Dynamic DNS supscription. If you want something cloud based, ASFA can be very helpful to you.

Key Features


  • Cloud Based Network: No need of DDNS (Dynamic DNS) provider to access your system. You can access your system anytime from anywhere. Only you need just an Internet Connection.


  • Telegram API Based: You can operate your system using Telegram Bots.


  • Command Line Interface: You can access your shell from Telegram. Its like an SSH connection.


  • Startup Application: It will run automatically during boot. No need to execute manually.


  • Security: No one can access your system when you are near.

Installation Steps


  • Create a Telegram Bot. You can do it using BotFather.




  • Download and Install the MSI file.


  • Go to C:\Program Files\Logic Realm\ASFA\_internal\ or Your_Installation_Folder\_internal.


  • Find api_key.txt and put your API Key inside.


  • Restart Your PC.

System Requirements


  • Operating System: Windows

  • Processor: 1.5 GHz or higher

  • Memory: 2 GB RAM

  • Storage: 30 MB available space
